# Task Manager App (React Native)

## Overview
This is a Task Manager mobile app built using React Native and Expo.  
It allows users to manage their daily tasks efficiently.

## Features
- Add, delete, and mark tasks as completed
- Search tasks by title
- Filter tasks (All / Completed / Pending)
- API integration (JSONPlaceholder)
- Local storage using AsyncStorage (data persists after restart)
- Clean and modern UI

## Tech Stack
- React Native (Expo)
- TypeScript / JavaScript
- AsyncStorage
- REST API

## How to Run

1. Clone the repository:git clone https://github.com/adityaashhhh/task-manager-react-native.git 
2. Install dependencies: npm instal
3. Start the app: npm start
4. Press `w` to run on web or scan QR with Expo Go.
